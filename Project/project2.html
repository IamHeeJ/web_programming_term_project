프로젝트 개요
  1. 서비스 개요
    • 두 사람의 채팅 내용을 AI 감정 분석하여 펫을 성장시키는 재미 요소와 목표 의식을 부여하여 연인 간의 다정한 대화 습관을 유도하는 커플 채팅 서비스
    • 서비스 명칭은 영단어 love와 breed를 합친 “러브리드(Luvreed)”로 선정

  2. 서비스 시나리오
    • 두 사용자가 채팅을 입력하면 실시간으로 AI 감정을 분류
    • 누적된 행복 데이터로 오른쪽처럼 펫이 성장
    • DB에 저장된 통계 데이터로 왼쪽처럼 감정 분석 통계와 피드백 제공
    • 관리자는 건의를 처리하고 답변을 작성하면 사용자에게 전달
    • 사용자와 관리자는 프로세스와 상관없이 게시판을 통해 언제든 건의글 조회

  3. 데이터 전처리
    • AI 허브에서 “연속적 대화”, “감정 분류 대화“ 데이터셋 두가지를 수집
    • 아래 예시처럼 각 문장 당 감정이 레이블링된 데이터셋으로 구성
    • 전처리에 앞서 두 데이터셋의 레이블을 통합하여 최종 레이블 7가지(중립,행복,분노,놀람,슬픔,짜증,불안) 선정
    • 모든 데이터는 중복 값과 공백을 제거하는 정제 과정을 수행
    • 그래프 불균형을 해소하기 위해 ‘연속적 대화 데이터 셋’ 에서 데이터가 부족한 레이블 4개(‘행복’, ‘짜증’, ‘불안’, ‘놀람’)를 통합해서 보충
    • 학습을 충분히 수행할 수 있도록 데이터 개수를 늘리기 위해 데이터를 증강
      • Random Masking Replacement : 무작위로 문장의 단어를 비슷한 문맥의 단어로 변경
      • Random Masking Insertion : 무작위로 문장의 문맥에 맞는 단어 추가
    • 채팅 대화체, 호칭, 단답형 단어, 비속어, 은어 등의 문장을 직접 레이블링한 커스텀 데이터 셋을 추가

  4. AI 모델 학습
    • 모델 학습에 앞서 다음 한국어에 특화된 4개 언어 모델(koBERT, kcBERT, koRoBERTa, koELECTRA)을 테스트
    • 테스트 결과, 정확도는 koELECTRA가 93%, koBERT가 87% 순으로 높았고, 예측 수행 결과 또한 두 모델이 비교적 우수한 성능을 보임
    • 모델 선정 기준으로 Accuracy와 Predict 체감 성능, 서버 배포의 용이성을 고려
    • 최신형 모델로서 BERT기반 모델에 비해 같은 학습량으로도 정확도가 높고, 예측 수행이 빨라 채팅 앱 특성에 적합한 koELECTRA를 서비스에 활용할 모델로 선정

  5. 펫 경험치 시스템
    • 회원가입 시 초기 사용자에게 4개의 알 기본 지급
    • 성장시킬 펫을 선택하여 “행복” 빈도수로 경험치를 높여 성장
    • 펫은 대화 감정 분류에 따라 3가지(긍정, 중립, 부정) 상태 표시
    • 감정 레이블이 ‘행복’일 때 경험치가 2씩 증가
    • 펫의 성장단계는 경험치 값에 따라 5단계로 구성

  6. 통계 시스템
    • 통계 시스템은 두 사용자 각각의 감정 분석 결과를 바탕으로 일일, 일주일, 한달 통계 3가지 통계 시스템 제공
    • 전날 하루의 전체 감정 레이블 중 행복 레이블의 비율을 기준으로 4가지(좋음,양호,나쁨,데이터부족) 피드백 제공
    • 일주일, 한달 동안의 행복 레이블의 개수의 평균 변화율을 계산하여 3가지(증가,감소,불안정) 피드백 제공
